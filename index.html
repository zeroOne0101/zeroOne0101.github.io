学习笔记-委托

发表于 2017-01-20   |  

委托

委托只是一种特殊类型的对象,其特殊之处在于,我们以前定义的所有对象都包含数据 ,而委托包含的只是一个或多个方法的地址

作用：传递方法

声明委托

定义好委托后,就可以创建它的一个实例，所创建的委托
的实例仍称为委托。

使用委托

简单的委托示例

类1：Math0perations.cs

类2：Program.cs

Action和Func委托

泛型Action委托表示引用一个void返回类型的方法。
Func委托可以以类似的方式使用 。Func允许调用带返回类型的方法

上面的示例改为以下代码，结果一样 


class Program
{
    delegate double DoubleOp(double x);
    static void Main(string[] args)
    {
        Func<double,double>[] operations =
        {
            Math0perations.MultiplyByTwo,
            Math0perations.Square
        };

        for(int i=0;i<operations.Length;i++)
        {
            Console.WriteLine("Using Operations[{0}]", i);
            ProcessAndDisplayNumber(operations[i], 2.0);
            ProcessAndDisplayNumber(operations[i], 7.94);
            ProcessAndDisplayNumber(operations[i], 1.414);
        }
        Console.ReadLine();
    }

    static void ProcessAndDisplayNumber(Func<double,double> action,double value)
    {
        double result = action(value);
        Console.WriteLine("Value is {0},result of operation is {1}", value, result);
    }
}
