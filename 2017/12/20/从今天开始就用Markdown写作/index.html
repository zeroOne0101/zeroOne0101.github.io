从今天开始就用Markdown写作

发表于 2017-12-20

Markdown 从之前就有了解，可以很方便于写作，通过一些简单的符号，设置不同的漂亮的适合写作的样式。但可能比较麻烦的就是图片的插入啦，需要找个地方上传，生成链接或者是本地路径，但只能本地查看。网上也有很多插件可以对图片进行上传，有需要的可以自己百度。
另外，今天发现Markdown真的很强大，还可以方便的生成图表，如流程图、甘特图等以及生成数学公式，真的太强大了。
